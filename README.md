# 法语数字练习 (French Numbers Practice)

一个用于训练法语数字听力、拼写和数字反应能力的 Web App。

## 主要功能

- **6 种练习模式**：看数字写法语、听语音写数字、看法语写数字、选择题（看数字选法语/听语音选数字）、快速反应
- **5 级难度系统**：入门 (0–20)、基础 (0–69)、进阶 (0–100)、困难专项、自定义
- **TTS 语音播放**：使用浏览器 Web Speech API，支持慢速/标准/快速
- **智能判分**：宽松的法语拼写匹配，支持大小写、连字符、空格等变体
- **错题本**：自动记录错题，支持复习、排序、删除
- **统计系统**：今日/总计正确率、连续答对、平均反应时间、按模式/难度统计
- **数据持久化**：使用 localStorage 保存所有进度数据
- **响应式设计**：适配手机和桌面端
- **键盘快捷键**：Enter 提交、Space 播放、N 下一题

## 练习模式说明

| 模式 | 说明 |
|------|------|
| 🔢 看数字写法语 | 看到阿拉伯数字，输入法语单词 |
| 🎧 听语音写数字 | 听 TTS 播放法语数字，输入阿拉伯数字 |
| 📝 看法语写数字 | 看到法语单词，输入阿拉伯数字 |
| ✅ 选择题（看数字选法语） | 看到数字，从 4 个选项中选正确法语 |
| 🔊 选择题（听语音选数字） | 听语音，从 4 个选项中选择正确数字 |
| ⏱️ 快速反应 | 限时完成听写/识别，测试速度 |

## 难度等级说明

| 等级 | 范围 | 说明 |
|------|------|------|
| 入门 (Niveau 1) | 0–20 | 适合刚学法语数字 |
| 基础 (Niveau 2) | 0–69 | 不含 70–99 |
| 进阶 (Niveau 3) | 0–100 | 包含所有数字 |
| 困难专项 (Niveau 4) | 易混淆数字 | 16/17/18/19, 60-99 |
| 自定义 (Niveau 5) | 自定义 | 自定义范围 |

## 安装方式

```bash
git clone <repo-url>
cd french-numbers
npm install
```

## 启动方式

```bash
npm run dev
```

### Windows

开发模式可双击 `start-windows.bat`，首次运行会自动安装依赖，并打开：

```text
http://127.0.0.1:5173/
```

App 版本可运行：

```bash
npm run build:win
```

生成的程序位于：

```text
release/win-unpacked/FrenchHelper.exe
```

Windows App 使用 Electron/Chromium 的 Web Speech API 播放法语语音。若无声音，请在 Windows 系统设置中安装或启用法语语音包。

## 浏览器 TTS 限制说明

- 本应用使用 `window.speechSynthesis` API 进行法语语音播放
- 需要现代浏览器支持 (Chrome、Edge、Safari 14+、Firefox)
- Safari 需要在系统设置中下载法语语音包
- iOS Safari 可能需要在用户交互后才允许播放语音
- 如果浏览器不支持 TTS，页面会显示友好提示
